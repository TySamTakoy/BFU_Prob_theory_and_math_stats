# ЗАДАЧА №3

С помощью python сделать анимации графиков биномиального распределения в зависимости от количества элементов n (сходимость к 0) для некоторого фиксированного значения вероятности p в Google Colab

# РЕШЕНИЕ

      sc.stats.binom(n, p).pmf(x)
      # n - количество испытаний
      # p - вероятность успеха в одном испытании
      # x - количество успехов
      # pmf - функция массы вероятности (probability mass function — это
      функция, которая возвращает вероятность того, что дискретная случайная
      величина примет конкретное значение. В контексте биномиального
      распределения, binom.pmf(x) возвращает вероятность получить ровно x
      успехов в n испытаниях с вероятностью успеха p.)
