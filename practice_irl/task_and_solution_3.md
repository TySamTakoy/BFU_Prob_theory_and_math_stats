# ЗАДАЧА №3

Вероятность изготовления на автоматическом станке стандартной детали равна $0,8$. Найти вероятности возможного числа появления бракованных деталей среди $5$ отобранных.

# РЕШЕНИЕ

Согласно схеме Бернулли:
$P_{m,n}=C_n^m⋅p^m⋅q^{n-m}$

Тогда $q=0,8$, $p=0,2$, $n=5$

Решим схему Бернулли для $m∈[1;5]$:

$P_{0,5}=C_5^0⋅0,2^0⋅0,8^{5-0}=0,32768$

$P_{1,5}=C_5^1⋅0,2^1⋅0,8^{5-1}=0,4096$

$P_{2,5}=C_5^2⋅0,2^2⋅0,8^{5-2}=0,2048$

$P_{3,5}=C_5^3⋅0,2^3⋅0,8^{5-3}=0,0512$

$P_{4,5}=C_5^4⋅0,2^4⋅0,8^{5-4}=0,0064$

$P_{5,5}=C_5^5⋅0,2^5⋅0,8^{5-5}=0,00032$


```
import matplotlib.pyplot as plt

probabilities = [0.32768, 0.4096, 0.2048, 0.0512, 0.0064, 0.00032]
x = range(len(probabilities))

plt.plot(x, probabilities, marker='o', linestyle='-')
plt.xlabel("m")
plt.ylabel("Pm,n")
plt.title("Биномиальное распределение (n=5, p=0,2)")
plt.xticks(x)
plt.grid(True)
plt.show()
```

$P_{1,5}=0,4096$ — это наивероятнейшее $m_0$ - число наступелния события

Тогда $m_0$ можно определить как:

$n⋅p-q≤m_0≤n⋅p+p$
