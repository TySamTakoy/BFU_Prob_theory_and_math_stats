# ЗАДАЧА №4

Сколько раз необходимо подбросить игралную кость, чтобы наивероятнейшее выпадение тройки было равно $10$?


# РЕШЕНИЕ

Воспользуемся выводом из предыдущей задачи:

$n⋅p-q≤m_0≤n⋅p+p$

Тогда с условием нашей задачи ($p=\frac{1}{6}$, $q=\frac{5}{6}$, $m_0=10$):

$n⋅\frac{1}{6}-\frac{5}{6}≤10≤n⋅\frac{1}{6}+\frac{5}{6}$

Тогда решив это неравенство, получим следующий ответ:

$n∈[59;65]$
