# ЗАДАЧА №5

На факультете насчитывается $1825$ студентов. Какова вероятность того, что $1$ сентября является днем рождения одновременно $4$ студентов факультета?

# РЕШЕНИЕ

Согласно теореме Пуассона:

$P_m(λ)=\frac{λ^m⋅e^{-λ}}{m!}$
$(λ=n⋅p≤10)$

Тогда с условием нашей задачи ($n=1825$, $p=\frac{1}{365}$, $m=4$):

$λ=1825⋅\frac{1}{365}=5≤10$

$P_m(λ)=\frac{5^4⋅e^{-5}}{4!}=0,175$
