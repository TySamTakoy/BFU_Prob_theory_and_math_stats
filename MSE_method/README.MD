# MSE method через математику и полностью на рандоме, только ввод кол-ва n.

Чем выше n, тем точнее построение вокруг исходного уравнения прямой, которое тоже в рандоме.

Задаем диаграмму рассеяния через рандом и отыскиваем график и уравнение через MSE method.

`Введите требуемое кол-во экспериментов:`

`50`

`Исходные данные: f(x) = -0.752 x + -7`

`Коэффициенты полученные МНК: f_mnk(x) = -0.77 x + -6.391`
