# ЗАДАЧА №2

Для задачи 4 были представлены решения, задействующие различные функции обучаемости: Кусочно-заданная (Куклевский Ярослав) Экспоненциальная из однопараметрического семейства (Алещенко Владислав) Дробно-экспоненциальная из двухпараметрического семейства (Рябов Артём) Интерполяция сплайнами (Климов Олег) Построить: Функции обучаемости (все 4 на одном графике; на графике должны быть также указаны точки  (0,0),(1,0.33),(10,0.95) ; в ячейке кода рядом с графиком требуется вычислить среднеквадратическое отклонение фактических значений каждой функции в указанных точках от требуемых по условию) Результирующие решения – зависимости мат. ожидания числа выученных билетов от количества билетов, выбранных для изучения (все 4 на одном графике)
